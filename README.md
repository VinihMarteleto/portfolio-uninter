# Portfólio Pessoal - Vinicius Marteleto

Este projeto foi desenvolvido como parte de uma atividade acadêmica da UNINTER, com o objetivo de criar um portfólio pessoal utilizando tecnologias web básicas.

---

## 📌 Objetivo

Desenvolver um site pessoal apresentando informações sobre mim, minha formação acadêmica, meus projetos e formas de contato.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (básico)

---

## 📄 Estrutura do Projeto

O site contém as seguintes seções:

- Sobre Mim
- Formação Acadêmica
- Portfólio
- Contato
